typofixer
=========

Какие опечатки могут быть:
1. пропущен символ
2. присутствует лишний символ
3. изменен символ
4. неправильная раскладка (возможно часть фразы, например "java ghjuhfvvbcn" -> "java программист")
5. слитное написание
